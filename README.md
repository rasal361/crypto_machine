# 🔐 Crypto Machine

A simple Python program to **encode and decode messages** using a basic character shift.

---

## 🚀 Features

* Encode messages
* Decode messages
* Beginner-friendly

---

## 🧠 How It Works

* Uses a fixed set of characters: `a-z`, space, `!`
* Shifts characters by one position
* Uses dictionaries for encoding and decoding

---

## ▶️ How to Run

```bash
python crypto_machine.py
```

---

## 📌 Example

```
Input: hello world
Mode: e
Output: ifmmp!xpsme
```

---

## ⚠️ Note

* Supports only lowercase letters, space, and `!`
* For learning purposes only (not secure)
